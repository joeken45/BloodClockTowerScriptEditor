=============================================
  血染鐘樓劇本編輯器 v0.0.1
  Blood on the Clocktower Script Editor
=============================================

【系統需求】
✓ Windows 10 或更新版本
✓ 64位元作業系統
✗ 不需要安裝 .NET Runtime（已包含）

【安裝方法】
1. 解壓縮此 ZIP 檔案到任意資料夾
2. 雙擊執行 BloodClockTowerScriptEditor.exe

【首次使用】
1. 點擊「檔案」→「開啟新檔」建立新劇本
2. 點擊「新增角色」按鈕匯入角色總表
3. 開始編輯你的劇本

【主要功能】
✓ 視覺化角色編輯
✓ 支援 BOTC 官方格式
✓ 劇本匯入/匯出
✓ 夜晚順序管理
✓ 相剋規則設定
✓ Special 特殊功能編輯

【資料夾說明】
- Data/ - 資料庫檔案（首次執行會自動建立）

【注意事項】
⚠ 請勿刪除 Data 資料夾
⚠ 建議定期備份劇本 JSON 檔案
⚠ 如遇問題請到 GitHub Issues 回報
